Test: Basic Slotted Options

A <select> inside a shadow root should recognize <option> elements slotted in from the light DOM.

1. Single-level slotted options

Options are in the light DOM of <my-custom-select>, slotted through a <slot> into the shadow DOM <select>.

Shadow DOM Light DOM (slotted) <slot> 
<my-custom-select>
  ── shadow root ──────────────────
  <select>
    <slot>
      ▼ assigned nodes (light DOM):
      <option value="a">Alpha</option>
      <option value="b">Beta</option>
      <option value="c">Gamma</option>
    </slot>
  </select>
  ─────────────────────────────────
</my-custom-select>

FLAT TREE (what select should see):
<select>
  <option value="a">Alpha</option>
  <option value="b">Beta</option>
  <option value="c">Gamma</option>
</select>

2. Pre-selected option

The second option has the selected attribute. select.selectedIndex should be 1.

Shadow DOM Light DOM (slotted) 
<my-custom-select>
  ── shadow root ──────────────────
  <select>
    <slot>
      <option value="x">X-ray</option>
      <option value="y" selected>Yankee</option>  ← selected
      <option value="z">Zulu</option>
    </slot>
  </select>
  ─────────────────────────────────
</my-custom-select>

EXPECTED:
select.selectedIndex === 1
select.value === "y"

3. Dynamic option insertion

A new option is added to the light DOM after initial render. The select should pick it up.

Shadow DOM Light DOM 
BEFORE (initial):
<my-custom-select>
  <select>
    <slot>
      <option value="1">One</option>
      <option value="2">Two</option>
    </slot>
  </select>
</my-custom-select>

AFTER click (JS appends to light DOM):
<my-custom-select>
  <select>
    <slot>
      <option value="1">One</option>
      <option value="2">Two</option>
      <option value="3">Three</option>  ← new
    </slot>
  </select>
</my-custom-select>

EXPECTED:
select.options.length: 2 → 3

4. Dynamic option removal

Removing a slotted option from the light DOM should update the select's option list.

Shadow DOM Light DOM 
BEFORE:
<my-custom-select>
  <select>
    <slot>
      <option value="r">Red</option>
      <option value="g">Green</option>  ← removed
      <option value="b">Blue</option>
    </slot>
  </select>
</my-custom-select>

AFTER click (JS removes from light DOM):
<my-custom-select>
  <select>
    <slot>
      <option value="r">Red</option>
      <option value="b">Blue</option>
    </slot>
  </select>
</my-custom-select>

EXPECTED:
select.options.length: 3 → 2
